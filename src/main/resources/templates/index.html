06/01/2025 - Search Page Update:

KWIX is nearing completion! Thank you for the support up until now. As of this update I would say things are 95% complete. I say only 95% because I am a bit of a perfectionist. Aside from small issues that you probably won't even notice, all of the core features are done. From here on out there may be some User Interface changes and small fixes here and there.

What is new then? The Search page.

This will allow users to query for specific records. Employees, Payroll, and Jobsites allow for real-time filter but the search page will allow to look up records across all 3 pages in one place. Why was this included? The filters on each individual page are merely quick searches, the Search page will allow for more robost searches. For example you could query the database to show all paystubs for a specific jobsite. That could be pretty useful. You can create all kinds of Search combinations on the Search page.

Read the Search page Documentation.
